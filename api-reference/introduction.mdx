Welcome to the Glyde Public API, your gateway to seamless financial transactions. This API provides a set of robust and secure endpoints designed to help you initiate payments, manage collections, verify customer information, and perform various financial operations with ease.

With the Glyde API, you can:
- Initiate collection requests to receive payments securely.
- Verify customer details, including bank account information.
- Check wallet balances and monitor transactions.
- Initiate bank transfers for disbursements.
- Retrieve a list of supported banks for seamless financial operations.
- Manage settlements efficiently.
