This section provides an interactive API reference where you can explore and test all Glyde API endpoints directly from the documentation.

## Base URLs

| Environment | URL |
|-------------|-----|
| Production | `https://api.useglyde.co/v1` |
| Sandbox | `https://sandbox.useglyde.co/v1` |

## Authentication

All API requests require a Bearer token in the Authorization header:

```
Authorization: Bearer sk_your_secret_key
```

## Using the API Playground

Each endpoint page includes an interactive API playground where you can:

1. View request and response schemas
2. Test API calls with your credentials
3. See example requests in multiple languages

<Tip>
  Use your sandbox API key when testing in the playground to avoid affecting production data.
</Tip>
